# Feb Internship Logic Building Task - Solutions

## Overview
This repository contains solutions for the Feb 2026 Data Science Internship Logic Building Task with 10 Python programming problems.

## Problems Solved

1. **Problem 1: Unique Words in a Sentence** - Find and count unique words in a given sentence using sets
2. **Problem 2: Highest Salary from Employee Data** - Identify the employee with the highest salary from a dictionary
3. **Problem 3: Find Maximum and Minimum Values** - Find max and min values from a list of numbers
4. **Problem 4: Count Products Above a Price Threshold** - Count products with price above a specified threshold
5. **Problem 5: Calculate Attendance Percentage** - Calculate percentage attendance from class data
6. **Problem 6: Remove Duplicate Phone Numbers** - Remove duplicate phone numbers from a list using sets
7. **Problem 7: Count Character Frequency** - Count frequency of each character in a string
8. **Problem 8: Convert List to Tuple** - Convert a list data structure to a tuple
9. **Problem 9: Check if a Key Exists in a Dictionary** - Verify key existence in a dictionary
10. **Problem 10: Calculate Average Marks** - Calculate average marks from a list of marks
